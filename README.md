# 🧬 ProteinAI — Agentic Drug Discovery Platform

A world-class futuristic AI biotech landing page built with React + Vite + Tailwind CSS.

## ✨ Features

- **10 fully animated sections**: Hero, Problem, Agentic AI, Features, Scientific Viz, Pipeline, Impact, Research, Testimonials, CTA
- **Custom cursor** with glow effect
- **Scroll progress bar**
- **Animated particle system**
- **Protein molecule SVG visualization**
- **Interactive Recharts** (Bar, Area, Line, Radar, Scatter charts)
- **Multi-agent orchestration visualization**
- **Glassmorphism cards** with hover effects
- **Gradient text and neon glow effects**
- **Intersection Observer** scroll reveals
- **Animated counters**
- **Responsive mobile layout**
- **Custom scrollbar**
- **Animated grid background**
- **DNA helix and orbital animations**

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```
